Linux® est un système d'exploitation (OS) open source. Un système d'exploitation est le logiciel qui gère directement le matériel et les ressources d'un système, comme le processeur, la mémoire et le stockage. Le système d'exploitation se situe entre les applications et le matériel et établit les connexions entre tous vos logiciels et les ressources physiques qui font le travail.

MacOS a été introduit en 1984 pour exécuter des ordinateurs personnels (PC) Macintosh. Le système d'exploitation a contribué à rendre les interfaces utilisateur graphiques (GUI) populaires, incitant même Microsoft à s'éloigner de sa méthode de saisie utilisateur en faveur de l'utilisation d'interfaces graphiques pour les systèmes d'exploitation Windows. Depuis octobre 2021, la version actuelle de macOS est macOS 12 Monterey.
